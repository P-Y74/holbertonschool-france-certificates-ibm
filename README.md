# IBM Certifications – Holberton School

## Description

This repository contains the IBM certifications completed during the **first year (Foundations)** at Holberton School.

These certifications are part of the official curriculum and are designed to reinforce key industry skills such as:

- open source fundamentals
- agile methodologies
- databases and SQL
- project management
- cloud computing
- cybersecurity fundamentals

They provide a structured introduction to professional tools, workflows, and concepts used in real-world environments.
